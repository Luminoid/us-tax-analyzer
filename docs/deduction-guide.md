# Deduction Guide

## Itemized Deductions (Schedule A)

### SALT — State and Local Taxes

**2025 Cap**: $40,000 MFJ / $20,000 MFS (phase-down for high AGI, floor $10,000)

| Tax Type | Deductible? |
|----------|-------------|
| State income tax (W-2 box 17) | Yes |
| Local income tax (W-2 box 19) | Yes |
| State disability insurance (W-2 box 19) | Yes |
| Property tax (real estate) | Yes |
| Vehicle license fee (VLF portion of car registration) | Yes |
| DMV fees (non-VLF portion) | No |
| State sales tax (alternative to income tax) | Yes |

**Phase-down (2025)**: If AGI exceeds $500,000 ($250,000 MFS), the cap reduces by 30 cents per dollar over the threshold, but not below $10,000.

**NRA note**: SALT is deductible on 1040-NR Schedule A, but with the same cap.

### Mortgage Interest

| Loan Origination | Debt Limit (MFJ/Single/HOH) | Debt Limit (MFS) |
|------------------|----------------------------|-------------------|
| On or before Dec 15, 2017 | $1,000,000 | $500,000 |
| After Dec 15, 2017 | $750,000 | $375,000 |

- Only **acquisition indebtedness** (buy, build, substantially improve)
- HELOC interest deductible only if used for home improvement
- If loan exceeds limit, deduction is proportionally reduced
- Multiple 1098 forms (loan transferred mid-year) — combine all
- **Not available on 1040-NR** (NRA filers cannot deduct mortgage interest)

### Charitable Contributions

| Type | Limit |
|------|-------|
| Cash to public charities | 60% of AGI |
| Cash to private foundations | 30% of AGI |
| Property (FMV) to public charities | 30% of AGI |
| Property to private foundations | 20% of AGI |

- Gifts of $250+ require written acknowledgment
- Non-cash gifts over $500 require Form 8283
- Carryforward available for amounts exceeding limits (5 years)

### Medical and Dental Expenses

- Deductible amount = expenses exceeding **7.5% of AGI**
- Includes insurance premiums (not pre-tax), prescriptions, dental, vision, medical travel
- Not available on 1040-NR

### Casualty and Theft Losses

- Only from **federally declared disasters** (post-TCJA)
- Subject to $100 per-event floor and 10% of AGI threshold

## Above-the-Line Deductions (Schedule 1)

These reduce AGI regardless of whether you itemize:

| Deduction | Limit | Notes |
|-----------|-------|-------|
| IRA contributions (traditional) | $7,000 ($8,000 if 50+) | Phase-out if covered by employer plan |
| Student loan interest | $2,500 | Phase-out at $80K–$95K single, $165K–$195K MFJ |
| HSA contributions | $4,300 single / $8,550 family | Must have HDHP |
| Self-employment tax (50%) | Half of SE tax | Only for self-employed |
| Self-employed health insurance | Premiums paid | Only for self-employed |
| Educator expenses | $300 | K-12 teachers only |

## Standard Deduction (2025)

| Status | Amount |
|--------|--------|
| Single | $15,000 |
| MFJ | $30,000 |
| MFS | $15,000 |
| HOH | $22,500 |
| Additional (65+ or blind) | +$1,600 single / +$1,300 married |

**Not available to**: NRAs on 1040-NR (except students from India under treaty), MFS filers if spouse itemizes.

## Itemized vs Standard Decision

| Your Situation | Likely Better |
|----------------|--------------|
| Homeowner with mortgage > $15K + high-tax state | Itemize |
| Renter in low-tax state, no big deductions | Standard |
| Large charitable giving year | Itemize |
| NRA filer | Must itemize (no choice) |
| One MFS spouse itemizes | Both must itemize |
