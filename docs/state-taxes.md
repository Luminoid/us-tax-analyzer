# State Tax Reference

## States by Income Tax Type

### No State Income Tax
Alaska, Florida, Nevada, New Hampshire*, South Dakota, Tennessee*, Texas, Washington, Wyoming

*New Hampshire and Tennessee tax only investment income (dividends/interest).

### Flat Rate States
Arizona, Colorado, Georgia, Idaho, Illinois, Indiana, Iowa, Kentucky, Michigan, Mississippi, Missouri, Montana, North Carolina, North Dakota, Ohio, Oklahoma, Pennsylvania, Utah, Virginia, West Virginia, Wisconsin

### Progressive Rate States (10%+ top rate)
California (14.4%), Hawaii (11%), New Jersey (10.75%), New York (10.9%), Oregon (9.9%), Minnesota (9.85%), Vermont (8.75%), District of Columbia (10.75%)

## Key State Rules

### California
- Top rate: 14.4% (including 1% Mental Health Services Tax over $1M)
- SDI (State Disability Insurance) withheld from wages — deductible as SALT on federal return
- Community property state — married MFS filers must split community income 50/50
- Does not conform to federal SALT cap
- Nonresidents/part-year: File Form 540NR

### New York
- Top rate: 10.9% (state) + up to 3.876% (NYC)
- Resident if domiciled in NY or maintained permanent abode + 184 days
- Does not conform to federal SALT cap
- Nonresidents/part-year: File Form IT-203

### Texas / Florida / Washington / Nevada
- No state income tax return required
- Property tax and vehicle registration fees still deductible as federal SALT
- Sales tax can be deducted instead of income tax as an alternative SALT deduction

## General State Considerations

### Residency
- State residency is determined independently from federal status
- Based on domicile, physical presence, or statutory rules (varies by state)
- You can be an NRA for federal purposes but a state resident for state purposes

### Filing Status
- Usually follows federal filing status
- Some states allow different status than federal in specific situations

### Multi-State Filing
- File nonresident return in each state where you earned income
- File resident return in home state (usually with credit for taxes paid to other states)

### State Tax Refunds
- If you **itemized** federal deductions and deducted state tax, a state refund in the next year is taxable federal income (tax benefit rule)
- If you took the standard deduction, the refund is not taxable
- Report on federal Schedule 1, line 1

### Community Property States
California, Arizona, Idaho, Louisiana, Nevada, New Mexico, Texas, Washington, Wisconsin

Married couples filing MFS must split community income 50/50 and file Form 8958.
